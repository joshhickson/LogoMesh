Executive Summary

The meeting served as a project review and onboarding session between Josh (project originator) and Deepti (joining as senior data scientist and technical program manager). 1. Core Project Concept

"Working with a really experienced coder who has dementia" - Josh's description of AI agents with Contextual Debt

Contextual Debt Definition:

  • The tendency for AI agents to "forget" foundational intent or the "why" behind code
  • Discovered after initial AI-built application failed due to unmanageable technical debt
  • Requires complete restart, leading to restructuring for AgentX competition

2. The Three Measurement Pillars

  1. Rational Debt: Scored via rubric evaluating hallucinating requirements or citing irrelevant context
  2. Architectural Debt: Quantitative measurement using escomplex library (cyclomatic complexity, maintainability index)
  3. Testing Debt: Quantitative score from code coverage and test pass/fail rates in secure sandbox

Note: Final weighting for combining scores was not yet determined at this meeting.
