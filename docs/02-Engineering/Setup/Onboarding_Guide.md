## 2. Onboarding Checklist

1.  **Set up the Environment:**
    *   `pnpm install` (Root)
    *   `cd onboarding && pnpm install`

2.  **Understand the "Why" (Strategy):**
    *   Read [PROJECT_PLAN.md (Outdated)](../../Archive/PROJECT_PLAN.md) (The Vision).
    *   Read [../../Archive/Unsorted/20251119-Strategic-Master-Log.md](../../../docs/Archive/Unsorted/20251119-Strategic-Master-Log.md) (The Pivot).

3.  **Understand the "What" (Architecture):**
    *   Run the visualization: `cd onboarding && pnpm start`.
    *   Explore the graph at `http://localhost:3000`.

## 3. The Visualization

The code in this directory runs a D3.js force-directed graph of our documentation.

*   **Data Source:** `src/data/doc_graph_raw.json` (Generated by `scripts/generate_doc_graph.js`)
*   **App Logic:** `src/main.ts` (Vite + D3)

## 4. Troubleshooting

*   **"Missing Links":** If the graph looks empty, run `node scripts/generate_doc_graph.js` from the project root to regenerate the JSON.
*   **"Dangling Edges":** Run `node scripts/analyze_dangling_edges.js` to find broken links.
